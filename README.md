# First things first: #
## Get your fresh composer.phar ##
`curl -sS https://getcomposer.org/installer | php`

## Create conposer.json ##

```json 
{
    "require": {
        "metalmini/silex-rest-service": "dev-master"
    }
}
```

## Run composer to install ##

`./composer.phar install`
